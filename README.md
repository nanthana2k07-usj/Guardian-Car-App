# 🚗 Guardian Car App

Guardian Car App is an AI-powered smart vehicle security and monitoring application designed to enhance vehicle safety through real-time monitoring, secure authentication, and intelligent theft prevention. The platform enables users to register their vehicles, manage emergency contacts, receive instant security alerts, and monitor their vehicle status from anywhere.

---

## 📌 Features

- Secure User Registration
- Email OTP Verification
- User Authentication
- Vehicle Registration and Management
- Emergency Contact Management
- Interactive Dashboard
- Real-Time Vehicle Alerts
- Theft Detection Notification System
- Profile Management
- Mobile Application Support (Android)
- Responsive User Interface
- Cloud-Based Data Storage

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/nanthana2k07-usj/Guardian-Car-App.git
```

### Navigate to the Project

```bash
cd Guardian-Car-App
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the project root and add the required environment variables.

Example:

```env
MONGODB_URI=your_mongodb_connection_string
SMTP_EMAIL=your_email
SMTP_PASSWORD=your_email_password
JWT_SECRET=your_secret_key
```

### Start the Development Server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```

---

## 📱 Android Application

Sync the Capacitor project:

```bash
npx cap sync
```

Open the Android project:

```bash
npx cap open android
```

Build and run the application using Android Studio.
